# umi-example-ssr

ssr example for umi, including dva.

## Usage

先安装依赖，

```js
$ yarn
```

然后运行 `yarn start`

在浏览器打开，

* [http://127.0.0.1:7001][2]

调试可使用 `yarn debug`，不会对代码进行压缩。
