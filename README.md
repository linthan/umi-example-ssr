# umi-example-ssr

ssr example for umi, including dva.

> 依赖 [umijs/umi!2543][1]。

## Usage

先安装依赖，

```js
$ yarn
```

然后分别运行 `yarn start` 和 `yarn server`。

在浏览器打开，

* [http://127.0.0.1:7001][2] - SSR
* [http://127.0.0.1:8000][3] - CSR
